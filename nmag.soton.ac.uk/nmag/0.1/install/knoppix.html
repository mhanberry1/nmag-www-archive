Running (Installing) nmag from the Knoppix CD

Knoppix (http://www.knopper.net/knoppix/index-en.html) provides a Linux system that can be booted directly from CD. The nmag team has tailored a Knoppix CD so that the nmag system can be tested easily (without having to install Linux). The CD should boot on most Intel PC hardware.

This 'installation' should be as easy as booting from the CD.

An iso image for the Knoppix CD is available in this directory.

Note that saving data needs some extra effort (see http://groups.google.com/group/nmag-users/browse_thread/thread/4f26ceddef9100b3) as the Knoppix systems lives entirely in the RAM of the machine (and, by default, never uses the hard drive).

The current version of code on the Knoppix CD is nmag 0.1 beta (5878) from 14 Oct 2008. Note that this is not the most recent version of the code.
